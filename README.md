# 🚀 Space Shelter Data Analysis

Proyecto de análisis de datos y Machine Learning
usando FastAPI + React + Scikit-Learn.

## 📦 Tecnologías

- FastAPI
- React
- TypeScript
- TailwindCSS
- Scikit-Learn
- Pandas

## 📊 Datasets

### 🚀 Space Launch Dataset
- Launches
- Rockets
- Agencies
- Success prediction

### 🐾 Shelter Animal Dataset
- Shelter outcomes
- Adoption prediction

## ----------------- ⚙️ Instalación  -----------------

### Backend

```bash
    cd backend
    pip install -r requirements.txt
    uvicorn main:app --reload
```
### Frontend
```bash
    cd frontend
    npm install
    npm run dev
```
